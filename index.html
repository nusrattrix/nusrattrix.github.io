Powerful Features

Advanced Music System

High-quality music playback with playlist support, lyrics, and crossfade. Supports YouTube, Spotify, SoundCloud and more.

Leveling & Economy

Complete leveling system with rank cards, leaderboards, titles, and virtual economy with shops and trading.

Moderation Suite

Advanced moderation tools including auto-mod, offense tracking, warning system, and comprehensive logging.

Welcome System

Customizable welcome and leave messages with embedded images, roles assignment, and member verification.

AI Chat System

Intelligent chat responses, context-aware conversations, and custom AI personalities for your server.

Fully Customizable

Every feature can be customized through our web dashboard. Change prefixes, commands, and behaviors easily.

Command Categories

/play Music

Play music from YouTube, Spotify, or direct URLs

/play <song name or url>
/queue Music

View and manage the music queue

/queue [page]
/skip Music

Skip the currently playing song

/skip
/warn Moderation

Issue a warning to a member

/warn @user [reason]
/clear Moderation

Clear messages from a channel

/clear <amount>
/ban Moderation

Ban a user from the server

/ban @user [reason]
/rank Leveling

Check your or someone's rank card

/rank [@user]
/leaderboard Leveling

View server level leaderboard

/leaderboard [page]
/bg Leveling

Change your rank card background

/bg <background name>
/serverinfo Utility

Display detailed server information

/serverinfo
/help Utility

Get help with Riri's commands

/help [command]
/userinfo Utility

Get information about a user

/userinfo [@user]
/meme Fun

Get a random meme

/meme
/8ball Fun

Ask the magic 8-ball a question

/8ball <question>
/cat Fun

Get a random cat picture

/cat
